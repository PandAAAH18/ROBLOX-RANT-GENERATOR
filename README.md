# Roblox Rant Generator

A Python application for creating Roblox rant-style videos with text-to-speech (TTS), subtitles, and gameplay backgrounds.

## Features

- **Text-to-Speech Generation**: Uses Microsoft Edge TTS for high-quality voiceovers.
- **Word-Level Customization**: meticulous control over pitch and rate for individual words to create dynamic and funny emphasis.
- **Subtitle Generation**: Automatically generates word-by-word subtitles with bounce animations.
- **Visuals**: 
  - Supports background gameplay videos.
  - Allows inserting images/memes at specific timestamps.
- **Templates**: Save and load voice setting presets (e.g., "High Pitch", "Slow").
- **Video Export**: Renders final videos using MoviePy (supports GPU acceleration if available).

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/PandAAAH18/ROBLOX-RANT-GENERATOR.git
    cd ROBLOX-RANT-GENERATOR
    ```

2.  **Create a virtual environment (optional but recommended):**
    ```bash
    python -m venv venv
    # Windows
    venv\Scripts\activate
    # macOS/Linux
    source venv/bin/activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Install ImageMagick (Required for MoviePy TextClips):**
    - Download and install [ImageMagick](https://imagemagick.org/script/download.php).
    - Ensure `magick` is in your system PATH.
    - For Windows users, if MoviePy complains, you might need to configure the binary path in `moviepy/config_defaults.py` or set the `IMAGEMAGICK_BINARY` environment variable.

## Usage

### GUI Application

Run the main application interface:

```bash
python vsub_tts.py
```

1.  **Input Text**: Type or paste your script into the "Input Text" area.
2.  **Parse**: Click "Parse Text" to break it down into sentences.
3.  **Customize**:
    - Select a sentence to edit.
    - Click individual words to adjust pitch (e.g., +50Hz) and rate (e.g., -50%).
    - Right-click words to add images/memes.
    - Use templates for quick styling.
4.  **Generate Audio**: Click "Generate Audio" to create the TTS file and timestamp data.
5.  **Generate Video**: Click "Generate Video" to render the final MP4.

### CLI Video Generation

If you have a configuration JSON file (generated by the GUI), you can render the video directly:

```bash
python generate_video.py path/to/config.json
```

## Project Structure

- `vsub_tts.py`: Main GUI application for editing scripts and generating audio/config.
- `generate_video.py`: Handles the video rendering process using MoviePy.
- `assets/`: Directory for background videos and meme images.

## Requirements

- Python 3.8+
- FFmpeg (usually installed automatically with moviepy, but system-wide install recommended for robustness)
